# 🧠 AI Novel Factory - 智能小說創作工廠使用者指南

歡迎來到 **AI Novel Factory (天衍小說創作工廠)**！這是一個專門為小說家、編劇和創意工作者打造的 **多智能體協同創作系統 (Multi-Agent Collaborative System)**。

系統整合了多位專業 AI 專家與一位協同總監，透過直覺的網頁介面，為您提供「靈感 ➡️ 世界觀 ➡️ 角色設計 ➡️ 大綱章節 ➡️ 內文寫作 ➡️ 精細編輯 ➡️ 檔案匯出」的全鏈路自動化與半自動化寫作體驗。

---

## 🤖 1. 核心 AI 協同智能體 (Agents) 介紹

本系統由 **5 位專門領域的 AI 專家** 與 **1 位協同創作總監 (Director)** 組成，每個 Agent 均擁有獨特的寫作定位與設定：

| 智能體名稱 | 核心角色定位 (Role) | 工作職責與特色 | 推薦溫度原則 |
| :--- | :--- | :--- | :--- |
| **Story Architect**<br>故事結構架構師 | 🗺️ 世界觀構建大師 | 負責規劃小說的**世界觀底層設定**、勢力範圍、力量體系、歷史脈絡與核心衝突。將您的零散靈感轉化為架構嚴密的世界觀設定集。 | **低溫度 (0.3 - 0.35)**：<br>需要精準、架構性的設定，避免設定前後矛盾。 |
| **Character Designer**<br>角色設計大師 | 👥 靈魂人物雕刻家 | 根據世界觀背景，精細化雕琢小說中所有核心角色。設計角色的姓名、身份、**性格標籤**、**致命缺陷**、**核心動機**與**人物成長曲線**，並輸出成角色卡片。 | **中低溫度 (0.45)**：<br>在具備創意的同時，保持人物設定的結構化。 |
| **Plot Planner**<br>章節劇情規劃師 | ⏳ 黃金分割大綱師 | 將世界觀與角色動機融合，自動將整本小說拆分為結構合理的**章節劇情大綱**。設計每一章的標題、核心情節事件、**寫作目的**、**伏筆與鋪墊**與**情緒基調**。 | **低溫度 (0.35)**：<br>要求邏輯嚴謹的大綱拆解與伏筆規劃。 |
| **Chapter Writer**<br>小說正文寫作作家 | ✍️ 文字具象魔術師 | 根據當前選中章節的大綱事件、世界觀與人物 Bible，進行極具文學張力的**正文 (Prose) 創作**。支援**深度思考模式**，在撰寫前進行深度的推理思考，讓情節銜接與人物對白更符合人性與張力。 | **中高溫度 (0.6 - 0.65)**：<br>創意寫作需要較高的發散度，以產生豐富的文學表達。 |
| **Editor Agent**<br>精緻文風編輯 | 🔍 文字拋光雕刻師 | 對已撰寫的章節正文進行**微調、精修與文筆潤色**。您可以給予它特定的精修指令（如：「加強動作打鬥的緊湊感」、「讓環境氣氛顯得更肅殺」），它會針對文字細節進行二次加工。 | **極低溫度 (0.25)**：<br>精準的文字微調，防止大幅偏離原文。 |
| **Co-pilot Director**<br>AI 總監 Copilot | 💬 隨身智囊 & 導演 | 常駐於右側側邊欄 of AI 對談小助手。它共享當前小說的資料庫與記憶，能隨時與您針對特定情節進行腦力激盪、解答創作瓶頸，或提供角色命運的全新點子。 | **中溫度 (0.55)**：<br>平衡創意建議與對話的連貫性。 |

## 🎬 2. 小說創作完整流程

系統採用 **四階段漸進式大綱生成策略** 與 **滾動式正文寫作管線**，整個創作流程如下：

```
使用者靈感輸入 (Copilot)
       │
       ▼
【Stage 1：前期準備】 ──→ 產生世界觀設定 (Story Architect) & 角色聖經 (Character Designer)
       │
       ▼
【Stage 2：宏觀骨架】 ──→ 定義各篇卷 (Volume) 結構、時間軸、篇卷摘要與規則
       │
       ▼
【Stage 3：伏筆編織】 ──→ 全局伏筆 (Foreshadowing) 埋設與回收節點對齊
       │
       ▼
【Stage 4：微觀大綱】 ──→ 滾動式生成詳細章節大綱（每批 5 章遞進生成）
       │
       ▼
【Stage 5：正文寫作】 ──→ Chapter Writer 根據大綱撰寫正文（可啟用 AI 深度思考過程）
       │
       ▼
【Stage 6：精緻編輯】 ──→ Editor Agent 針對特定段落或整章進行文筆精修與拋光
```

### 總監決策循環 (Director Loop)
在您與右側的 **Copilot 總監** 互動時，AI 總監會持續監控當前創作狀態，並自動採取以下決策：
- **CONTINUE**：自動推進至下一個創作階段（例如從世界觀設定進入篇卷骨架）。
- **LOCAL_ALIGN_VOLUME**：針對單一卷進行即時 (JIT) 設定微調與校準。
- **INCREMENTAL_INSERT_PLOT**：在大綱中增量插入新的情節章節。
- **GO_BACK_TO_***：回退修改（如回退至修改世界觀設定、修改角色 Bible 或重新調整大綱）。
- **AUTO_REGENERATE**：遇到生成不符預期時，自動重新嘗試生成。

## ⚙️ 3. 模型設定與 API Key 配置

在網站左下角點擊 **「⚙️ 模型設定 & API Key」**，即可進入 Agent 的配置面板。

### 前端設定步驟
1. **選擇 Agent**：點擊設定面板左側的導航按鈕選擇您要配置的 Agent。
2. **編輯設定**：您可以自訂該 Agent 使用的 API Key、API Base URL、模型名稱、溫度 (Temperature)、Top P、以及 Max Tokens。
3. **快速套用預設值 (Nvidia Presets)**：點擊選單可一鍵套用 NVIDIA 推薦的頂尖模型設定：
   - `nvidia/nemotron-3-super-120b-a12b`：120B 推理模型，開啟思考模式表現卓越，適合**正文寫作**與**總監決策**。
   - `openai/gpt-oss-120b`：極致強大的通用寫作大模型，適合**角色設計**與**大綱編排**。
   - `mistralai/mistral-small-4-119b-2603`：119B 高推理、高智慧模型，適合**大綱與正文精修**。
   - `minimaxai/minimax-m2.7`：精緻的情感細節與文學描寫，適合**情感戲與細節描寫**。
   - `google/gemma-3n-e4b-it`：Gemma 3n 精校版，適合快速處理與文字潤色。
   - `stepfun-ai/step-3.5-flash`：極速響應，適合**側邊欄快速對話 (Copilot)**。
4. **啟用深度思考 (Reasoning / Thinking)**：
   > [!NOTE]
   > 若選用支援 Reasoning 的模型（如 Nemotron 思考版），勾選「**啟用深度思考**」後，系統在寫作正文時會輸出詳細的推理心路歷程，這能大幅提升複雜情節的邏輯連貫性。
5. **儲存設定**：點擊「**儲存本項設定**」將設定寫入本地資料庫。

## 📦 4. 小說資料匯出指南

當小說創作完成或達到階段性成果時，您可以將小說的所有數據匯出為本地檔案。

### 支援匯出格式對照

| 格式 | 用途與特點 | 檔案內容結構 |
| :--- | :--- | :--- |
| **TXT 格式**<br>(純文字) | 適合直接閱讀、分享給他人或導入電子書閱讀器。 | 僅包含：<br>1. 書名、題材、風格基調<br>2. 排序好的各章章節標題與正文內容 |
| **Markdown 格式**<br>(結構化) | 適合做二次排版、轉換為 PDF/EPUB，或保留完整的創作設定作為備份。 | 包含完整的創作數據：<br>1. 小說元資訊<br>2. **世界觀與核心設定**<br>3. **角色聖經 (Character Bible)** (JSON 格式)<br>4. **劇情章節大綱** (JSON 格式)<br>5. 排序好的完整章節正文 |

### 如何在網頁端匯出
1. 開啟 AI Novel Factory 網頁介面。
2. 進入 **Writer (寫作)** 標籤頁。
3. 在上方工具列（"AI 撰寫本章正文" 等按鈕旁），點擊 **「匯出小說」** 按鈕。
4. 選擇您要的格式（**TXT 格式** 或 **Markdown 格式**）。
5. 瀏覽器將會自動下載生成的檔案，檔名會自動採用您的小說標題（例如：`{小說標題}_完整正文.txt` 或 `{小說標題}_小說設定與全書正文.md`）。

## 一、目前流程檢查結論

### 1. 傳入資料是否齊全

以最新小說《最後一班地鐵》為檢查對象：

- 世界觀完整：已建立「暮影列車」「時間流紋」「車票詛咒」「列車指揮官」「逆流之咒」等核心設定。
- 角色資料完整：最新角色聖經有 9 名角色，包含周牧、黎曦、林曉、阿爾塔、沈影、張景元、艾莉絲、卡爾、幽霧。
- 篇卷資料完整：共 9 卷，計畫章數 413 章。
- 卷骨架完整：9 卷的 `volumes.chapters_outline` 都已生成，合計 413 章，沒有缺卷骨架。
- 正文目前已有 181 章。第 180 章與第 181 章都接在「光之門 / 第三條路 / 祖父即列車指揮官」的高潮點。

需要注意：

- `plot_chapters` 舊表目前只有 10 筆，但系統已註明此表 deprecated；實際 `get_stitched_plot()` 會優先縫合 `volumes.chapters_outline`，所以不應用舊表數量判定大綱缺漏。
- 正文已寫 181 章，但總規劃是 413 章。若要繼續第一部，應寫第 182 章；若要開第二部續作，應新建 sequel pipeline prompt，不要直接覆寫原小說資料。

### 2. 上下文是否會太多

目前已有幾個好的控管：

- Director 最近對話只抓 1 則，避免聊天記憶污染決策。
- Writer 只拿前一章與後一章大綱，並另外拿前後卷摘要。
- Writer 會依 `characters_active` 過濾角色，只傳當章活躍角色。
- Writer review 只保留前章尾巴、當章正文、後章參考，不吃全書正文。
- 伏筆審核改用 Python 預計算分配表與 `allocated_tasks`，避免總監臨時通靈。

主要風險：

- `prompts/prompt_builder.py` 中 `MAX_WORLDVIEW_SUMMARY_LENGTH`、`MAX_MACRO_OUTLINE_LENGTH`、`MAX_CHARACTERS_SUMMARY_LENGTH` 都設為 999999，等於取消摘要上限。長篇作品後期很容易讓 character、copilot、director prompt 變胖。
- `run_copilot_chat()` 仍會保留最近 10 則聊天記憶，若使用者長談設定，可能把非當前任務資訊帶入。
- `run_director_decision_help()` 的調閱模式會拉完整世界觀、完整角色或完整大綱，適合人工求助，不適合自動循環高頻使用。
- `run_chapter_writer()` 目前沒有傳入前一章正文尾巴，只傳前後章大綱。若要提升章章銜接感，應加入「前章最後 800-1200 字」作為 writer input，而不是只在 review 階段使用。

### 3. 每個 agent 收到的資料是否清楚

- Story Architect：清楚，輸入為類型、風格、使用者需求，輸出 JSON schema 明確。
- Character Designer：清楚，但世界觀摘要實際不再壓縮，長篇世界觀會過重。建議只傳「世界觀摘要 + 角色相關規則 + 已通過伏筆遮罩版」。
- Volumes Planner：清楚，適合全書規劃；但目前 generate 模式會在少於 8 卷時自動補足到 8 卷，短篇/中篇需求可能被硬拉長。
- Volume Skeleton Planner：清楚，輸入包含當卷、前後卷、預計算伏筆分配；非常適合並行化。
- Chapter Writer：大方向清楚，且角色過濾有效；但缺少前章正文尾巴，容易造成語氣與現場狀態斷裂。
- Editor：清楚，但如果只給原文，不給當章大綱與角色摘要，可能過度文風化。review 階段有比較完整，但 edit agent 本身可再補。
- Director：清楚且有剛性校驗報告；但它仍會受到長 prompt 影響，應把「決策用狀態」改成更短的機器摘要。

## 二、提升 agent 並行能力的建議

### 可立即並行的任務

1. 卷骨架生成可改成並行批次：
   - 每次並行 2-3 卷。
   - 每卷只讀完整世界觀摘要、當卷、前後卷、伏筆分配。
   - 完成後做一次全局連續性校驗。

2. 微觀章綱可用滑窗並行：
   - 以 5 章為一批，每批重疊 1 章上下文。
   - 例如批次 A：1-5，批次 B：5-9，批次 C：9-13。
   - 重疊章只作為錨點，不重複寫入，最後由 merge 層消歧。

3. 正文可有限並行：
   - 不能無腦全章並行，因為章尾銜接很重要。
   - 可採「場景草稿並行，章節定稿串行」：同章多場景由 writer 子任務草擬，再由 lead writer 合併。
   - 或採「相隔較遠章節試寫並行」，例如只並行寫大綱已穩定且互不依賴的支線章。

4. 角色補完與伏筆審核可與正文寫作並行：
   - Writer 生成第 N 章時，Critic 可預審第 N+1 至 N+3 章大綱缺口。
   - Continuity Agent 可維護一份「上一章狀態摘要」，供下一章直接使用。

## 三、第二部接續小說提案

### 書名

《最後一班地鐵 II：第四道門》

### 核心命題

第一部結尾中，周牧沒有選擇封印，也沒有選擇逆流，而是打開第三條路。第二部追問的是：當一個人拒絕宿命後，世界是否真的會放過他？如果「選擇」本身也能被利用，那自由意志還剩下什麼？

### 故事承接

周牧在光之門中消失後，城市表面恢復正常。失蹤者陸續回到原本的人生，但他們都遺失了一段共同記憶：終點站之後的七分鐘。

林曉接手一個被官方抹除的秘密小組，專門追查「回歸者後遺症」。沈影沒有死，她成了終點站與現實之間的半透明信使，只能在雨夜、霧中與舊地鐵廣播裡留下訊息。

三個月後，一名年輕外送員許知遙收到一張沒有目的地的車票。車票背面不是「請交出你的名字」，而是另一句話：

「請替周牧作證。」

### 新主角

許知遙：二十二歲，外送員，父親是第一部失蹤回歸者之一。她不相信神秘事件，只相信訂單、路線和錢包餘額。她的能力不是看見靈魂，而是能聽見「未被選擇的聲音」。

### 回歸角色

- 周牧：不再是現實中的人，而是第三道門內的守門人。他只能透過車票裂縫傳遞片段。
- 林曉：時間犯罪小組最後的留守者，既想救周牧，也害怕打開新裂縫。
- 沈影：半消失狀態，成為最危險也最可靠的嚮導。
- 黎曦：列車指揮官的人格殘片，可能是祖父留下的保護機制，也可能是第四道門的誘餌。
- 阿爾塔：放棄永生後成為流亡祭司，掌握「見證儀式」。

### 新衝突

第一部的敵人想控制列車；第二部的敵人想控制「選擇」。

新勢力「校準局」宣稱要修復城市時間線，實際上正在收集每個人未被選擇的人生，把它們壓縮成一座名為「第四道門」的裝置。只要第四道門開啟，所有人都會被送回人生中最關鍵的選擇點，重新作出一次選擇。但代價是：第一次選擇留下的人生會被整條抹除。

### 三幕式大綱

第一幕：回歸者的七分鐘

許知遙收到車票，父親開始夢遊到已封鎖的地鐵站。林曉追查多起回歸者同時失語事件，發現所有人都在紙上寫下同一句話：「周牧沒有死，他被判有罪。」

第二幕：未被選擇的人生

許知遙能聽見不同人生的自己在耳邊說話。她與林曉、沈影合作，進入殘存車廂，發現周牧在第三道門內被迫審判：他打開的可能性太多，導致城市時間線不再唯一。校準局試圖讓周牧承認第三條路是錯誤，藉此關閉所有自由分歧。

第三幕：第四道門

校準局啟動第四道門，全城所有人回到一個關鍵瞬間。許知遙必須決定：救周牧，會讓城市再次陷入時間亂流；放棄周牧，則證明自由意志只是系統允許的幻覺。最終她選擇不替任何人重選人生，而是讓每個人記住自己曾經選過什麼。第四道門崩解，周牧第一次以真正的人類身份回到雨夜月臺。

## 四、第二部開篇正文

### 第一章：沒有目的地的車票

凌晨零點四十四分，城市的雨聲像一卷被反覆播放的舊磁帶。

許知遙把最後一份宵夜掛在門把上，手機螢幕顯示訂單完成。她站在老公寓的騎樓下，把安全帽摘下來，甩了甩短髮上的水。雨滴順著外送箱的邊角往下淌，霓虹燈在積水裡碎成一條條發抖的紅線。

她討厭這座城市的雨。

不是因為冷，也不是因為濕，而是因為每次下雨，她父親就會坐在客廳裡，盯著沒有打開的電視，像在等一班永遠不會進站的車。

手機又震了一下。

新的訂單跳出來，取餐地點顯示為：「暮影站地下二層」。

許知遙皺起眉。這城市沒有暮影站。至少在她跑外送的三年裡，導航地圖上從沒有這個地方。

她本想按掉，卻看見配送費後面多了一個離譜的數字。足夠她付掉父親這個月的藥錢，還能把那台總是半夜自己響起的舊收音機扔掉，換一台不會嚇人的。

「詐騙也該有點誠意吧。」她低聲說。

話音剛落，手機畫面忽然閃了一下。取餐地點消失，變成一句黑底白字的提示：

請於零點四十九分前抵達月臺。

逾時者，將被視為放棄作證。

許知遙的指尖停在螢幕上。

街角的燈同時暗了半秒。

她抬起頭，看見巷口不知何時起了霧。霧很低，貼著地面流動，像從城市的下水道裡慢慢吐出來。遠處傳來一聲鐘響，空洞、遙遠，明明附近沒有教堂，卻準確地在雨夜裡敲出了某種不屬於現在的時間。

一聲。

兩聲。

三聲。

她忽然想起父親清醒時唯一說過的那句話。

不要撿雨裡的車票。

許知遙低頭。

一張泛黃的車票靜靜躺在她的外送箱蓋上。雨水落在票面，卻沒有浸濕它。票面沒有起點，沒有終點，只有一串淡得幾乎看不清的打孔數字。

112。

她的呼吸停了一拍。

就在這時，車票背面慢慢浮出一行字，像有人用看不見的手指在紙上寫字。

不是父親曾經反覆夢囈的那句「終點站到達時，請交出你的名字」。

而是另一句。

請替周牧作證。

許知遙盯著那個陌生名字，胸口卻莫名一緊，彷彿那兩個字不是第一次出現在她生命裡。她聽見耳邊有許多聲音同時低語，有女人的聲音，有老人的聲音，有小孩的哭聲，也有一個疲憊卻溫和的男人聲音。

他們說的不是同一句話。

可所有聲音都指向同一個方向。

月臺。

手機再次震動，導航自動開啟。地圖上的道路一條條熄滅，只剩一條藍色路線穿過雨霧，通往城市早已封閉的舊地鐵入口。

許知遙把車票捏在指間，紙面冰冷得像剛從某個冬天裡撈出來。

她應該把它丟掉。

她應該回家，看著父親吃藥，假裝這座城市從來沒有吞掉過任何人。

可她忽然想起三個月前，父親從失蹤中回來的那天。他坐在醫院病床上，眼神空得像被挖掉一塊時間。所有人都說他只是創傷後失憶，只有許知遙看見他掌心裡被指甲掐出的血痕。

那血痕歪歪扭扭，寫著一個字。

門。

雨越下越大。

許知遙把安全帽扣回頭上，跨上電動車。車燈刺破霧氣的一瞬間，她在後照鏡裡看見騎樓深處站著一道半透明的人影。

那是一個女人，穿著黑色風衣，像一片從夜裡裁下來的影子。

女人對她抬起手，食指抵在唇邊，示意她不要回頭。

然後舊收音機的雜訊聲從許知遙的手機裡響起，沙沙作響，像雨水流過鐵軌。

「聽好，」女人的聲音從雜訊裡傳來，「如果你真的要去暮影站，記住三件事。」

許知遙握緊車把。

「第一，不要相信月臺上的時鐘。」

藍色導航線在霧中延伸。

「第二，不要回答任何人問你的名字。」

遠處，地鐵入口的鐵門在無人觸碰的情況下緩緩升起。

「第三，」女人的聲音忽然變得極低，像怕驚醒什麼沉睡在地下的東西，「如果你看見周牧，不要救他。」

許知遙猛地抬眼。

「為什麼？」

雜訊沉默了一秒。

下一刻，一列黑色地鐵從霧中無聲滑入月臺，車窗裡沒有乘客，只有一排排倒映著雨夜的空座位。

女人的聲音再次響起。

「因為他現在，可能已經不是站在門裡的人。」

車門開了。

一陣溫暖的風從地下吹出來，帶著鐵鏽、雨水，以及某種很淡很淡的外送餐盒香氣。

許知遙低頭看見車票上的字又變了。

請於第四道門開啟前入座。

她笑了一聲，很短，像是在罵人，也像是在壓住恐懼。

「行。」她把車票塞進口袋，推著車往地鐵入口走去，「最好這單真的有人付錢。」

身後的城市時鐘，在同一秒全部停住。

而暮影站的月臺廣播，以一種溫柔得近乎殘酷的聲音響起：

「各位乘客，歡迎搭乘最後一班地鐵。」

「本班列車即將前往——未被選擇的明天。」
